Struct libtrig::Pos2D

source · 
#[repr(C)]
pub struct Pos2D { /* private fields */ }
Expand description

A 2D position with rotation.

This type is a wrapper around the Coord2D and Angle2D types.

All logical operations are implemented for this type.
